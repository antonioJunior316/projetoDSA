//feito em ambiente linux ubutun versão 24.04
//comandos basicos pra instala compilador c/c++

comando instala compilador = 
    sudo apt update
    sudo apt install build-essential

comando pra compilar e executa = 
    gcc main.c processos.c escalonamento.c metricas.c -o escalonador && ./escalonador


o modo de estrutura o projeto 

estrutura Processo(struct) para centralizar os dados e funções que são usadas pra trabalha sobre esses dados,separação por módulos pra melhora organização do codigo, uso de array pra fica facil manusear codigo sem risco de erros na mémoria e também melhor desempenho ja que os endereços na mémoria são proximos

critérios de desempate(Preemptivo)
quando dois processo tiverem a mesma prioridade e ja chegaram ao sitema, ordem de varredura pelo ID menor

criterios de desempate(Não Preemptivo)
quando mas de um processo esta podendo executa no mesmo momento os criterios sao menor prioridade de numero,ordem de chegada e ordem de cadastro menor ID

calculo das métricas

turnaround = tempo de termino menos tempo de chegada

waiting = turnaround menos a duração do processo

response =  tempo de inicio menos  tempo de chegada

Throughput = tempo total de todos os processos dividido pelo numero de processos
